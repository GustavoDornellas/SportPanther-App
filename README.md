# 🏀 SportPanther App

## 🧠 Sobre o Projeto

O SportPanther é uma aplicação mobile desenvolvida com o objetivo de conectar jovens atletas a oportunidades no esporte.

O projeto surgiu a partir da dificuldade que muitos jogadores enfrentam para obter visibilidade e acesso a oportunidades, como testes em clubes, campeonatos e contatos com olheiros.

A proposta é oferecer uma plataforma centralizada onde atletas possam divulgar seus perfis e aumentar suas chances de serem encontrados.

---

## 🎯 Problema

* Baixa visibilidade para atletas iniciantes
* Dificuldade de acesso a oportunidades esportivas
* Falta de uma plataforma centralizada para divulgação

---

## 💡 Solução

Uma aplicação mobile que permite:

* Criação e divulgação de perfis de atletas
* Centralização de informações esportivas
* Maior conexão entre atletas e oportunidades

## 🏗️ Arquitetura

O projeto foi estruturado de forma modular, separando responsabilidades entre frontend e backend:

* **Frontend mobile:** responsável pela interface, navegação e experiência do usuário
* **Backend:** responsável pela lógica de negócio e persistência de dados
* **Banco de dados:** armazenamento estruturado das informações dos usuários
