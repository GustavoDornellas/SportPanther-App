# ⚽ SportPanther App

## 🇧🇷 Sobre o Projeto

O **SportPanther App** é uma aplicação desenvolvida como Trabalho de Conclusão de Curso (TCC), com o objetivo de incentivar e ampliar a participação feminina no futebol. A plataforma conecta jogadoras e facilita o acesso a locais públicos para prática esportiva, promovendo inclusão, visibilidade e fortalecimento do futebol feminino.

## 🇺🇸 About the Project

**SportPanther App** is an application developed as a Final Course Project, aiming to encourage and expand women's participation in football (soccer). The platform connects players and facilitates access to public spaces for sports practice, promoting inclusion, visibility, and strengthening women's football.

## 🚀 Funcionalidades | Features

* 👥 Conexão entre jogadoras | Connection between players
* 📍 Localização de espaços públicos | Location of public spaces
* 📢 Incentivo à participação feminina | Encouragement of women's participation
* 🤝 Promoção de inclusão e acessibilidade | Promotion of inclusion and accessibility
